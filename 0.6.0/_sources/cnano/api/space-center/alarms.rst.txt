   .. function:: krpc_error_t krpc_SpaceCenter_AlarmManager_AddTransferWindowAlarm(krpc_connection_t connection, krpc_SpaceCenter_Alarm_t * result, krpc_SpaceCenter_Vessel_t vessel, krpc_SpaceCenter_CelestialBody_t target, const char * title, const char * description)

      Create an alarm for the next planetary transfer window from the vessel's
      current parent body to the target body.

      :Parameters:

       * **vessel** -- The vessel.
       * **target** -- The target body.
       * **title** -- Title for the alarm.
       * **description** -- Description for the alarm.

   

      .. note::

         This relies on KSP's stock transfer-window alarm logic. If KSP cannot
         compute a transfer from the vessel's current parent body to the target,
         the resulting alarm may not fire at a useful time; in that case the
         properties on the returned alarm can still be used to inspect or adjust it.

.. type:: krpc_SpaceCenter_AlarmType_t

   The type of an alarm. See :func:`krpc_SpaceCenter_Alarm_Type`.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_RAW

      An alarm for a specific date/time or a specific period in the future.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_APOAPSIS

      An alarm for the next apoapsis of a vessel.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_PERIAPSIS

      An alarm for the next periapsis of a vessel.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_MANEUVER

      An alarm based on a maneuver node on the vessel's flight path.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_SOICHANGE

      An alarm for the next sphere of influence change on the vessel's flight path.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_TRANSFERWINDOW

      An alarm for the next planetary transfer window from the vessel's current
      orbit to a target body.


   .. macro:: KRPC_SPACECENTER_ALARMTYPE_UNKNOWN

      The alarm is of a type not recognized by kRPC. Typically this is a type
      introduced by a mod.



.. type:: krpc_SpaceCenter_AlarmWarpAction_t

   The warp action taken when an alarm fires.
   See :func:`krpc_SpaceCenter_Alarm_WarpAction`.


   .. macro:: KRPC_SPACECENTER_ALARMWARPACTION_NOCHANGE

      Do not change time warp when the alarm fires.


   .. macro:: KRPC_SPACECENTER_ALARMWARPACTION_STOPWARP

      Drop out of time warp when the alarm fires.


   .. macro:: KRPC_SPACECENTER_ALARMWARPACTION_PAUSEGAME

      Pause the game when the alarm fires.



.. type:: krpc_SpaceCenter_AlarmMessageAction_t

   The on-screen message action taken when an alarm fires.
   See :func:`krpc_SpaceCenter_Alarm_MessageAction`.


   .. macro:: KRPC_SPACECENTER_ALARMMESSAGEACTION_NOMESSAGE

      Do not display a message when the alarm fires.


   .. macro:: KRPC_SPACECENTER_ALARMMESSAGEACTION_MESSAGE

      Display a message when the alarm fires.


   .. macro:: KRPC_SPACECENTER_ALARMMESSAGEACTION_MESSAGEIFNOTACTIVEVESSEL

      Display a message only if the alarm's vessel is not the currently active vessel.
