kRPC Java client

Java client for kRPC, a Remote Procedure Call server for Kerbal Space Program.
It lets you control the game from an external program over the network.

Documentation:
 * Java client documentation: https://krpc.github.io/krpc/latest/java/client.html
 * kRPC documentation:         https://krpc.github.io/krpc
 * Source and issue tracker:   https://github.com/krpc/krpc
