kRPC C++ client

C++ client for kRPC, a Remote Procedure Call server for Kerbal Space Program.
It lets you control the game from an external program over the network.

See INSTALL.txt for build and installation instructions.

Documentation:
 * C++ client documentation: https://krpc.github.io/krpc/latest/cpp/client.html
 * kRPC documentation:        https://krpc.github.io/krpc
 * Source and issue tracker:  https://github.com/krpc/krpc
