kRPC C-nano client

C client for kRPC, a Remote Procedure Call server for Kerbal Space Program. It
lets you control the game from an external program over the network, and is
intended for use on resource constrained environments such as Arduino and
Raspberry Pi.

See INSTALL.txt for build and installation instructions.

Documentation:
