Servo
=====

   :func:`ServoGroup::servos`,
   :func:`ServoGroup::servo_with_name`

.. enum-struct:: ServoMode

   The mode a servo is operating in. See :func:`Servo::mode`.


   .. enumerator:: servo

      The part acts as a servo, driving towards a target position.


   .. enumerator:: rotor

      The part acts as a rotor, spinning continuously.
