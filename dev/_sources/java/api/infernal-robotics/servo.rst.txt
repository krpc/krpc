Servo

.. type:: public enum ServoMode

   The mode a servo is operating in. See :meth:`Servo.getMode()`.


   .. field:: public ServoMode SERVO

      The part acts as a servo, driving towards a target position.


   .. field:: public ServoMode ROTOR

      The part acts as a rotor, spinning continuously.
